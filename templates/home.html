Home

This is a flask based web app that uses the bootstrap framework to deal with visuals. The purpose of the app is to search through a data base using SQLite and display the data in a easy to read way.

How to use

Click on the multisearch tab in the nav bar above to access the search page. You can search a spell/s by id, clas, etc... A spreedsheet that can be accessed in the nav bar above that contains all the spells available. Multiple queries can be done at once by seperating the searches with a "|" e.g. Bard|Cleric. The search type dropdown can be used to do search for multiple types at once such as searching for a spell with the class Bard and duration of 1 minute using "Bard|1 minute" with class and duration selected in the search type dropdown.
